 College Event Management System

A responsive web application developed using React, JavaScript, HTML, and CSS that enables students to explore campus events, register for events, and manage their registrations through an interactive user interface.

 Project Overview

The College Event Management System is designed to simplify the process of browsing and managing college events. It provides separate sections for viewing events, registering for events, managing registrations, and accessing an admin dashboard interface.

Features

 Home Page

* Event Search Functionality
* Event Cards with Images
* Event Details Modal
* Countdown Timer for Upcoming Events
* Dark Mode Toggle

Events Page

* View All Available Events
* Register for Events
* Form Validation
* Interactive Event Cards

 My Events Page

* View Registered Events
* Delete Registrations
* Local Storage Support

Admin Dashboard

* Admin Interface UI
* Event Management Layout

 Technologies Used

* React.js
* JavaScript (ES6+)
* HTML5
* CSS3
* React Router DOM
* Local Storage

 Installation and Setup

 Clone the Repository

```bash
git clone https://github.com/Manasapinjala/college-event-management-system.git
```

Navigate to the Project Directory

```bash
cd college-event-management-system
```

 Install Dependencies

```bash
npm install
```

### Run the Application

```bash
npm run dev
```

The application will be available at:


http://localhost:5173
